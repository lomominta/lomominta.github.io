Projecto 2

Creacion de una base de datos

El objetivo de este proyecto fue desarrollar una base de datos para almacenar la información de los estudiantes de una hogar infantil. La base de datos debe incluir información sobre el nombre del estudiante, nombres de los padres, el número de telefo de los padres, la direccion de residencia de los padres, el grado y el promedio de calificaciones. El alcance de este proyecto incluye el desarrollo de la base de datos, la creación de las tablas necesarias y la carga de los datos. El proyecto no incluye el desarrollo de una aplicación web para consultar la base de datos
